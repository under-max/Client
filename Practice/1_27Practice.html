    참고용 치트차트

  • CSS. Attribute

    • 위는 참고 표


    1.Attribute는 속성 혹은 특성

    2.Attribute 는 Element 가 가지고 있는 부가적인 정보

    3.모든 Element 는 attribute 를 가질수 있음

    4.Attribute는 시작 태그에 들어감

    *Ex) Attribute name(속성값) = “value”

    5.Attribute 는 2개든 3개든 들어갈 수 있음

    6.값이 없고 이름만 있는 Attribute 도 있음

    7.<a href="주소값(https://포함하여 넣어야 함)">하이퍼링크로 들어갈 이름 <a>

    8.<start tag + title = "입력 값"> = 입력값 안에 마우스 대면 내용이 확인됨

    img src = <"이미지 주소 값" width = "" height = "" alt = "이미지 출력 실패시 나타나는 메시지">

  • Style

    • 1. Element 에 스타일을 추가한다 (색상, 글꼴 사이즈(크기)등)

    ex)style = "color : blue; font-size : 40px"

    2.property : Style 속성의 값(들)

    텍스트 정렬: text-align : center;, text-align : end;, text-align : start;

    *Element - Attribute - property

  • Color

    • 1.color 의 표현 방법에는 4가지 방법이 있다

    color : rgb(0,0,0);

    color : red;

    color : #000000; (16진수 의미 255를 프로그래머용 계산기로 나누어 계산 하면 편리)

    color : rgba(0,0,0,0); 마지막은 투명도와 불투명도 0일수록 투명 1일수록 불투명
