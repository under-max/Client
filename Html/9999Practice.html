Title: 1월 26일 Html 내용

  • html

    • html는 Hyper Text Markup Language 의 약자

    Html의 Element는 3개의 태그로 이루어짐

    Start Tag, Content, End Tag

    이 한덩어리를 Element 라고 함

    Start Tag 와 End Tag의 종류 테그는 같아야 함

  • 문단용 리스트

    • <h1> = h1부터 있으며 최대 h6까지 있음 1일수록 큰글씨

    <preh1> = 작성한 그대로 보여주는 테그

    <li> = 텍스트 기울이기

    <p> = 문단용 일반 텍스트

    <hr> = 줄긋기 용도

    <br> = 엔터

    <a> = 링크 저장용

    <img> = 이미지 엘레멘트

  • 특수 기호 리스트

    • 스페이스 = &nbsp;

    & = &amp

    < = &lt;

    > = &gt;

    Ctrl + /
